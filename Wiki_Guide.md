This is a wiki guide on how to organize submissions to the appropriate catagory. 

 There will be set catagories for submissions those being, graphing, Statistcal processes, git hub, libraries. 
 
 Graphing would be for process of understanding what should be displayed onto graphs such as error bars, legends, color, etc.
 
 Statiscal processes is for processes such as chisq value which will have its out subsection which will include details such as redeuced chisq and fitting. Monte Carlo is another process that would be included. If other stat process are used feel free to add here. 
 
 Libaries : common libaries that we use such as pandas, numpy, matplotlib, etc. This section fill provide details on how to use these libaries, will including some more complex libaries such as astropy and Qutip.
 
 Git hub : Will be designed for anything related to git hub such as how to push, commit, etc.
