One thing I understood well in PH264 was how to plot experimental data with error bars to show uncertainty clearly. Plotting data with error bars is useful because it shows both the measured values and uncertainty at each point and turns a list of numbers into somehting we can interpret.
For example if you measure the voltage across a circuit multiple times, you might get slightly different values each time. Instead of just plotting one number, you can show the average value with erros bars to represent how the measurements vary. This makes it easier to see how reliable the data is and whether the differences are meaningful.
