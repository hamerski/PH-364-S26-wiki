I'm still kinda new to coding and computational phsyics especially, but before taking this class I had taken a few math courses that were more computational, them being Applied ODE's and then Numerical Linear algeba. The Applied ODE was mostly plotting solutions and wasn't the most coding intesive but for Numerical Linear algebra it definitely was. 

The main themes of Numerical Linear algebra was taking concepts from linear algebra and implmeneting them computationally. Instead of solving large systems by hand, we can write code that allows us to solve large systems computing things like eigenvalues,eigenvectors and singular value decomposition of matrices. This helped highlight the idea that computation allows us to extend mathematical techniques to very large system that would be hard to solve otherwise. 

An example of this computational theme is for Einsteins field equations are a set of nonlinear differential equations which cannot be solved using regular methods for solving differential equations. So the only way we can solve these equations are by applying numerical solutions and computational models to understand how the space-time coordinates are curved. 

So applying the same logic to computation physics that we did math, we can take some physical questions and apply some computational methods allowing us to better understand the model.
