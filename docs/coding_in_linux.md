## Concept Background: Coding in Linux

In undergrad, I did a little bit of computational quantum chemistry focusing on the band structure of thermoelectric semiconductors.  All of the coding was done through linux using a premade program. A lot of the coding used what we learned about coding in the terminal along with additional data input including atomic number, position, lattice vectors, and atomic size. A basic understanding of quantum chemistry is needed to understand many electron systems, band theory, and how electrons behave. The most important concept in my case was understanding band theory, looking for materials with specific  narrow band gaps- the energy gap between the highest occupied orbital and the lowest unoccupied orbital. As for coding, understanding the code is mainly creating an moving through directories (with the additional data insertion) helped me understand what was happening inside the computer.

## Concept Showcase: How to Navigate Directories with Linux

Linux is a powerful and efficient tool to navigate through file directories in your computer. Some basic commands that you will use frequently to navigate are as follows:
- ```cd```: Allows the user to navigate through file directories. Two important applications of this command are as follows: ```cd ..``` moves back one directory. ```cd {your/absolute/file/path/here}``` will navigate exactly to the directory specified no matter what directory the user is currently in.
- ```ls```: Displays all of the files and folders in the current directory.
- ```pwd```: Displays the path of the directory the user is currently in.
