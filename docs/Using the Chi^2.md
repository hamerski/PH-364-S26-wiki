One of the main themes from PH264 was how to use python as a means of exploring datasets to model different systems. One recurring topic was the minimization of the chi squared value of a function. This allowed us to find the best possible values for constants and coeffecients to tack onto a function and fit a dataset. This is a very useful tool as it allows us to create a model and extrapolate future behaviors for the system.
