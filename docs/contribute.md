# Contribution

1. Create a branch or fork the repo.
2. Add a Markdown page under docs/ or a notebook under docs/notebooks/.
3. Update mkdocs.yml → nav to include your new page, this will add a link to the document in the sidebar.
4. Run mkdocs gh-deploy in your local dev environment.
5. Commit & push your branch; open a Pull Request.
6. After your PR is merged, the site will be published.
