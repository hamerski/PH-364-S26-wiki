# Tags
- [plasmaPY]

PlasmaPy is a python module used mostly within plasma physics, but has a wide range of functions than can be used in particle physics, nuclear physics, and stellar astrophysics. The particles submodule is the main subpackage within PlasmaPy, giving the user the ability to read particle properties, find the energy of nuclear reactions, recombine and ionize particles, and create molecules.
